# ESP8266 Control servo and ESC in Browser

# Upload instruction

## Specification .bin files
  -  Module: Generic ESP8266 Module
  -  Flash Size: 1M(256K SPIFFS)
  -  CPU Frequency: 80Mhz
  -  Flash Mode: dio
  -  Flash Frequency: 40Mhz
  -  Upload Speed: 115200

- GPIO 12 - Relay
